JQuery-selector-attr/method

发表于 2016-12-28   |   分类于 前端 ， JQuery ， 选择器/属性/样式   |     |   阅读次数

介绍jQuery的基础语法，选择器以及jQuery的一些属性和样式

JQuery初识

JQuery是一个类型数组对象, 而DOM对象就是一个单纯的DOM元素

JQuery对象转化为DOM对象

DOM对象转化成JQuery对象

JQuery选择器

id选择器 $(‘#id’)
类选择器 $(‘.class’)
元素选择器 $(‘element’)
全选择器 $(‘*’)
层级选择器
  – 子选择器 $(‘parent > child’)
  – 后代选择器 $(‘ancestor descendant’)
  – 相邻兄弟选择器 $(‘prev + next’)
  – 一般兄弟选择器 $(‘prev ~ sibling’)

基本筛选选择器

选择器 描述
$(‘:first’) 匹配第一个元素
$(‘:last’) 匹配最后一个元素
$(‘:eq(index)’) 匹配集合中索引值为index的元素
$(‘:gt(index)’) 集合中大于index的元素
$(‘:even’) 索引为偶数的元素,0开始
$(‘:odd’) 索引为奇数的元素,0开始
$(‘:lt(index)’) 集合中小于index的元素
$(‘:header’) 所有标题元素 h1 h2 h3
$(‘:lang(language)’) 选择指定语言的所有元素
$(‘:root’) 选择该文档的根元素
$(‘:animated’) 选择正在执行动画效果的元素
$(‘:not(selector)’) 一个用来过滤的选择器, 选择所有去除不匹配给定选择器的元素

内容筛选选择器

选择器 描述
$(‘:contains(text)’) 选择所有包含指定文本的元素(匹配文本包含在元素的子元素中,同样认为符合条件)
$(‘:parent’) 选择所有含有子元素或者文本的元素
$(‘:empty’) 选择所有没有子元素的元素(包含文本节点)
$(‘:has(selector)’) 选择元素中至少包含指定选择器的元素

可见性筛选选择器

选择器 描述
$(‘:visiable’) 选择所有显示的元素
$(‘:hidden’) 选择所有隐藏的元素

几种方法隐藏一个元素

  1. CSS display:none
  2. type=’hidden’的表单元素
  3. 高宽显式设置为0
  4. 一个祖先元素是隐藏的, 该元素不会在页面上显示
  5. CSS visibility的值hidden
  6. CSS opacity的值为0

如果元素占据文档一定的空间,元素被认为是可见的.
可见元素的宽或高大于0
元素visibility:hidden,opacity:0被认为是可见的, 因为占据了空间

属性选择器

选择器 描述
$(‘[attribute¦=value]’) 选择指定属性值等于给定字符串或以该字符串为前缀(该字符串后跟一个连字符”-“)的元素
$(‘[attribute*=value]’) 选择指定属性具有包含一个给定子字符串的元素
$(‘[attribute~=value]’) 选择指定属性用空格分隔的值中包含一个给定值的元素
$(‘[attribute=value]’) 选择指定属性是给定值的元素
$(‘[attribute!=value]’) 选择不存在指定属性,或者指定的属性值不等于给定值的元素
$(‘[attribute^=value]’) 选择指定属性是以给定字符串开始的元素
$(‘[attribute$=value]’) 选择指定属性是以给定字符串结尾的元素,区分大小写
$(‘[attribute]’) 选择所有具有指定属性的元素
$(‘[attributefilter1][attributefilter2]’) 选择匹配所有指定属性筛选器的元素

子元素筛选器

选择器 描述
$(‘:first-child’) ==$(‘:nth-child(1)’) 选择所有其父元素下的第一个子元素
$(‘:last-child’) 选择所有其父元素下的最后一个子元素
$(‘:only-child’) 如果某个元素是其父元素的唯一子元素,那么它就会被选中
$(‘:nth-child(n)’) 选择所有其父元素的第n个子元素,从1开始计数
$(‘:nth-last-child(n)’) 选择所有其父元素的第n个子元素,倒数, 从1开始计数

表单选择器

选择器 描述
$(‘:input’) 选择所有的input/textarea/select和button元素
$(‘:text’) 匹配所有文本框, 前面建议加上input:text,下同
$(‘:password’) $(‘[type=password]’)密码框
$(‘:checkbox’) 复选框
$(‘:radio’) 单选框
$(‘:submit’) 提交按钮
$(‘:image’) 所有图像域
$(‘:button’) 所有按钮
$(‘:file’) 文件域

表单对象属性筛选选择器

| 选择器 | 描述 |
| $(‘:enabled’) | 选取可用的表单元素|
| $(‘:disabled’) | 选取不可用的表单元素|
| $(‘:checked’) | 选取被选中的input元素,前面加input|
| $(‘:selected’) | 选取被选中的option元素,前面加option|

特殊选择器this

this是JavaScript的关键字, 指的是当前上下文对象
这个上下文对象可以被动态的改变(call, apply),改变对象this指向的内容

在DOM中this指向了这个html元素对象, this是DOM元素本身的一个引用

var $this =$(this)
this, 表示当前的上下文对象是一个html对象,可以调用html对象的属性和方法
$(this), 表示上下文对象是一个JQuery对象, 可以调用对象的方法和属性值
